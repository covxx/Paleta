QuickBooks Setup Instructions
Before importing, ensure QuickBooks is properly configured:
  1. Environment Variables: Set the following environment variables:
    • QB_CLIENT_ID - Your QuickBooks app client ID
    • QB_CLIENT_SECRET - Your QuickBooks app client secret
    • QB_COMPANY_ID - Your QuickBooks company ID
  2. OAuth Setup: Complete the QuickBooks OAuth flow to get access tokens
  3. API Permissions: Ensure your app has permissions to read customers and items
  4. Sandbox vs Production: Currently configured for sandbox environment
Important Notes:
  • Import will update existing records if QuickBooks ID matches
  • New records will be created for items not found in the system
  • Only inventory items are imported (service items are skipped)
  • Import process may take several minutes for large datasets
